<h1>Blackjack</h1> 

Blackjack desarrollado en Java con interfaz gráfica Swing.  
Proyecto individual orientado a práctica de programación orientada a objetos (OOP) y desarrollo de interfaces gráficas.

---

Descripción

- Objetivo: superar la puntuación del crupier sin pasarse de 21.  
- Sistema de fichas: el jugador empieza con 10,000 fichas, gana o pierde según la partida.  
- Cartas del jugador visibles en pantalla.
- Presiona Pedir Carta para sumar cartas.
- Presiona Plantarse para dejar que el crupier juegue.
- Reinicio automático al presionar **ENTER** cuando termina la partida.

Cómo jugar

1. Clona el repositorio:
https://github.com/clemen13/BlackJack.git
2. Abre el proyecto en NetBeans.
3. Dentro del paquete game, ejecuta la clase BlackJackGame.java.
Presiona ENTER para reiniciar después de que termine la partida.
